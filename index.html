About

Twin Mini Golf is our OS project which uses sdl2, a general API that provides low level access to audio, keyboard, mouse, joystick, 3D hardware via OpenGL, and 2D framebuffer across multiple platforms.

Read more...

How to play:

To begin a shot, hold the left mouse button. Drag to adjust the direction and force of your swing!

Release the left mouse button to shoot, and try to reach each hole in as few swings as possible!

Hint: Each hole can be completed in a single swing!

WARNING

Don't play this game unless you have 16gb ram or more. This game has a very big memory leak and will stop responding and start crashing your computer very quickly. But you can complete the full course if you're fast enough before it gets laggy. This timer was not intended but it makes the game more challenging.
